About

🌻 hinatabokkuri

ゲーム・音楽・フリー素材を制作しているインディーゲームクリエイターです。

Godot 4 でミステリーサウンドノベル、レトロJRPG、Wizardry風ダンジョンRPGの3作品を同時開発中。素材・BGMは制作後にフリー配布予定。

楽曲は Suno Pro と ElevenLabs・AivisSpeech の TTS 合成。ジャケット画像は ComfyUI + RealVisXL や Gemini を使い分け。

Solo Full-Stack

作詞・作曲・MV・カバーアート・サイト構築・データ公開まで 全レイヤーを1人で運用しています。AIを実装の道具として使いつつ、企画・ジャンル選定・最終判断は本人が担当。

前職は Java エンジニア → 銀行員。22曲 + PWAサイト + JSON API をひとりで回しているのが、地味だけど自分の強みかなと思っています。
